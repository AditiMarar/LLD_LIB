# LLD_LIB
Python-based Library Management System\
Uses Object-Oriented Programming (OOP)\
Manages books, members, and transactions\
Supports issue and return of books\
Tracks book status: AVAILABLE / ISSUED\
Uses Enums for genre, book type, and status\
Separates Book and BookItem (physical copy)\
Maintains a Catalog for searching books\
Registers members with Library Cards\
Prevents issuing reference books\
Validates member and book availability\
Records issue date and return date
